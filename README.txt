Quantum Research Technologies website
=====================================

Files:
- index.html: main website structure
- styles.css: full visual design, white/blue identity, responsive layout
- script.js: directory, board, news data, mobile menu, search, animations
- assets/qrt-logo.jpeg: QRT logo

How to edit the content:
1. Open script.js.
2. Edit directoryMembers to add real directory profiles.
3. Edit boardMembers to add board names and roles.
4. Edit newsItems to update the News section.

How to use:
- Open index.html in a browser to preview locally.
- Upload the whole folder to GitHub Pages, Netlify, Vercel, or any static hosting provider.
